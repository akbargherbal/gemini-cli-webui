Can you write a simple JavaScript function to calculate the factorial of a number?
Certainly! Here is a simple recursive function to calculate the factorial of a number in JavaScript:
function factorial(n) { if (n === 0 || n === 1) { return 1; } return n * factorial(n - 1); } console.log(factorial(5)); // Output: 120

Alternatively, you can use an iterative approach if you are concerned about stack overflow for very large numbers.
